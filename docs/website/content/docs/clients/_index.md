Vrata is API-first. The REST API is the only interface — any tool that speaks HTTP can configure the proxy.

The repo includes official clients:

- **Kubernetes Controller** — watches Gateway API resources and syncs them to Vrata automatically
- **UI** — planned

Third-party clients can be built against the [OpenAPI spec](/docs/api-reference/endpoints/).
